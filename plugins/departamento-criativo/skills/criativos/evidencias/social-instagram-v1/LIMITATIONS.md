# LIMITATIONS — social-instagram-v1

- **Associação, não causa.** Nenhuma recomendação garante resultado.
- **Performance = likes + comentários (proxy)**, sem followers/ER. Popularidade != qualidade.
- **Nenhum par isola uma feature** → não atribuir desempenho a um atributo.
- **cover-only** = só a capa (sem slides/vídeo/áudio/retenção).
- **Controles contextuais** = referência analítica, não modelo.
- **Zero pt-BR de ER**; corpus fixo; sem análise nativa de vídeo.
- Exemplos são **ilustrativos**, nunca prova de regra.
