# 01 — COMPOSIÇÃO VISUAL

> Consultar ao compor qualquer layout de seção, ao definir framing de imagem/vídeo e ao estruturar a hierarquia visual de uma página.

---

## 1. Grid matemático como fundação invisível
- Baseline de **8pt** (ou 4pt) sustenta toda composição
- Sistema modular por razões definidas (**1.25 / 1.333 / 1.5**) — nunca tamanhos arbitrários
- Até layouts "quebrados" respeitam eixos da grid

## 2. Hierarquia por posição e escala — nunca por ornamento
- Tamanho, localização e peso carregam **100%** da importância
- Zero decoração pra chamar atenção (sem glows, drop-shadows gratuitos, bordas ornamentais)

## 3. Assimetria calculada como padrão
- Centralização só quando matematicamente/narrativamente justificada
- Off-center seguindo razões (⅓, ⅖, ⅗, ⅔)
- Peso balanceado por campos de texto e espaço vazio — não por objetos

## 4. Negative space é estrutura, não respiro
- Espaço vazio carrega informação (tempo, silêncio, hierarquia)
- Proporção espaço:conteúdo favorece o espaço
- Generoso externamente, rítmico internamente

## 5. Alinhamento rigoroso
- **Flush-left ragged-right por padrão** — nunca justified em web
- Eixos visíveis ou implícitos alinhando todos os elementos

## 6. Ponto focal único por vista
- Cada fold / frame / seção = **UMA** afirmação dominante
- "Se tudo é importante, nada é importante"
